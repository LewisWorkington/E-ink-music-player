# Mechanical Connections

## Enclosure

- Houses all electronics
- Protects display and battery
- Provides button access

## Mounting

- ESP32 mounted using standoffs
- Battery secured with adhesive pad
- Display mounted to front panel

## External Access

- USB charging port
- microSD card slot
- Headphone jack or speaker opening
- Navigation buttons

## Future Improvements

- Slimmer enclosure
- Improved cable management
- Waterproof design option
