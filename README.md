# 🎵 E-Ink Music Player

A low-power portable music player built around an ESP32 and E-Ink display.

## Features

- E-Ink Display
- microSD Music Storage
- Physical Navigation Buttons
- Low Power Consumption
- USB Charging
- Open Source Design

## Hardware

| Component | Purpose |
|------------|----------|
| ESP32 | Main Controller |
| E-Ink Display | UI Display |
| microSD | Music Storage |
| Audio DAC | Sound Output |
| Battery | Portable Power |

See repository folders for documentation, firmware and hardware design files.
